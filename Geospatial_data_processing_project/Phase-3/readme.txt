We have used a python script to run with three set of parameters:
	1. Number of cores (ranging from 1 to 3 per executor).
	2. Number of executors (ranging from 1 to 3).
	3. Executor Memory.

To run the script (sample command):
python script.py --cores 3 --memory 3G --executors 3
